FILAMENT
日誌

カナリア バーミリオン ピスタチオ ジュニパー ママレード ナス

　エントリー 1

到着は円滑でした。船員たちはまだぼんやりしていますが、体調も精神状態も良好です。健康管理官のバーミリオンは、即座に私たちに栄養を補給し、体を動かすよう促しました。素晴らしい気分です。バーミリオンは船での経験は浅いようですが、頼もしい存在となってくれるに違いありません。

ほかの船員も落ち着きつつあります。理解を深めるにはもう少し時間がかかりそう。でも、任務履歴に目を通し、確信を持ちました。きっと私たちは、偉業を成し遂げることができるでしょう。

今週は比較的ゆったりとしたスケジュールです。チーム内の信頼関係を深めることに集中します。私たちは、他のどんな会社からも遠い場所にいるのですから。

航海中に損傷を受けましたが、いずれも軽微で、ほとんどが外観上のものです。心配の必要はないでしょうが、適切な対処を期待します。

担当キャプテンはスワン。これは、喜ばしい驚きです。彼女の功績については耳にしたことがあります。少し話しましたが、良好な協力関係が築けるでしょう。

-----

今日は、船員へ期待する事と、この地で何を達成したいかについて打ち合わせを行いました。権威者としての立場を確立する初めての機会だったので、不安がありました。うまくいったと思います。あまり威圧的な印象を与えたくはありませんが、これは私がリーダーを務める上での最初の任務。軽んじられるわけにはいきません。

ジュニパーには少し手を焼くかもしれない、と既に予感があります。見たところ彼女は非常に有能ですし、私たちをここまで導いてくれたのは彼女の手腕です。ただし、主張が強い。結論を急がず、もう少し様子を見たいと思います。

-----

今朝、研究室でサボテンを見つけました。ピスタチオに連絡したところ、調査スペースを汚したお詫びの品として置いたそうです。なんて思いやりがあるのでしょう。毅然と対応したことを後悔しました。今後は良好にやっていけるはずです。

-----

ジュニパーについては、判断を保留するよう心掛けていますが、ここ数日の彼女の態度は難しいです。何といっても、彼女はTFCの手順を気にしません。キャプテンと少し話したところ、彼女も私を支持してくれました。後ほど、全員に再認識してもらうための会議を開きます。 この問題には一切妥協しません。彼女が納得いくまで、必要なだけ会議を開くつもりです。

-----

NO3- 0ppm
NO2- 0ppm
NH3 0ppm

注: 卵の孵化まで残り2〜4日。水草の健康状態は良好。肥料5mlを週2回。

　エントリー 2

もう1週間？ あっという間でした。立ち止まる暇もなかったように感じます。今のところ順調です。定期メンテはすべて計画通り。最後の難関は、今週後半に予定されている船外活動だけです。

困ったことに、船乗り猫のセーブルが行方不明になりました。不本意なほど船員の時間を奪っています。船でペットを飼う利点についてはまだ納得していませんが、TFCにはそれを正当化する十分なデータがあるのでしょう。

もしかしたら、犬じゃなかったことに反感を持っているだけかも。

-------

今日から調査開始。また仕事ができるのは良い気分です。すでに没頭しています。アーノルド-475Mは、私がこれまで観測したことのないものです。ナスとママレードは、アラバスターのすべての観測装置の設置を、満足いくまで手伝ってくれました。私は楽観視しています。

--------

奇妙な状況に陥っています。今朝、アラバスターの軌道減衰が発覚しました。心配するほど深刻なものではありません。すぐにアーノルドと衝突する危険性はありませんが、ジュニパーの能力に疑問符が付きます。原因究明できないようなら、スワンに連絡する必要があるかもしれません。 また、ゲームナイトの開催があるとのこと。

------

自分に失望した夜でした。恥ずかしい。

-----

船外活動の件、私の対応は性急でした。私はただ、事故が適切に処理されるか確認をしたかったのですが、船員の感情を考慮に入れませんでした。皆、特にジュニパーには、もう少し寛容に接しようと思います。これは、私が思い描く人間像ではありません。

ジュニパーは、実によくやってくれました。彼女がママレードと一緒でなければ、葬儀を開く事態になっていたかもしれません。船にいてくれて、本当に良かった。

-------

バーミリオンは、シンプルな食事で皆の精神的な不調を和らげるという、卓越した能力をまたも発揮しました。地球のロースト・ディナーです。最後にこんな食事をしたのはいつだったでしょう。

--------

NO3- 15ppm
NO2- 0ppm
NH3 0ppm

注: 全ての稚魚が正常に孵化。ブラインシュリンプの給餌を開始。小さな藻類が発生、肥料の投与量を減らすこと。

close
　エントリー 3

土曜日に、ママレードのサプライズ・誕生日パーティーを開くことになりました。胸が躍ります。先週のことがあったので、私も積極的に企画を手伝うことにしました。ささやかですが、関係改善の第一歩となればと思います。

--------

ここ数日は、特筆することはありません。いつも通りの業務です。先週から士気は少し上がっているようですし、ママレードも速やかに回復しています。ナスの仕事ぶりは見事で、セラピーセッションも効果的でした。皆、パーティーを楽しみにしています。彼はまだ何も知りません。

--------

ママレードのパーティーは大成功。彼は上機嫌でした。

-------

どうやらTFCは、船員の士気に関する実験を行っているようです。研究の一環として、船のペットが出産した場合、船員にどのような影響があるかを記録するとのこと。私たちも対象らしいのですが、馬鹿げているどころではありません。今のところ、真剣に受け止めているのはきっと私だけですね。

セーブルの出産までの期間を、バーミリオンに確認する必要あり。

--------

NO3- 10ppm
NO2- 0.3ppm
NH3 0ppm

稚魚の群れのアンモニア生成により、微量の亜硝酸塩が検出。解決のため水換え。窒素循環が安定するまで注視。稚魚の状態は良好。

　エントリー 4

ついに、私たちがここにいる理由。いわば、核心。アーノルド-475Mの地表に展開する、探査機任務に着手しました。開発・組立て・最終的な打ち上げにおいて、船員たちに役割を振りました。自信がみなぎっています。変わり者ぞろいですが、彼らは著しく技能に長けています。

-------

ナスを十分に称賛できていないかもしれません。彼は私の期待を常に超えてくれます。データ収集アレイに関する彼のシステム作業は、一流です。

-------

セーブルが出産しました。予想通り、生産性は急降下。特にアーノルドの地理分析については顕著です。でも、ピスタチオに厳しく言うつもりはありません。彼女は一貫して、比類なき熱意で実用的なデータを提供してくれています。

この猫の件で、私は少数派のようです。周りは非常に興奮しています。判断は保留にしましょう。生産性が回復する限り、大した問題ではありません。

-------

再び、ゲームナイトの開催。雰囲気はやや敵対的になりましたが、私はなんとか冷静さを保つことができました。ナスの2度目の勝利を目の当たりにし、なかなか難しいことでした。どうやら彼の役職の性質が、アドバンテージを与えているようです。

あるボードゲームについては禁止も考えました。しかし摩擦を引き起こすにもかかわらず、皆には愛されているようです。

探査機の開発は順調です。進捗は十分。このペースなら、打ち上げの数日前に完了するでしょう。ピスタチオは理想的な着陸地点を発見しました。

航行計画も完了。あとはデータ収集アレイを完成させ、探査機の船体にストレス・テストを行うだけです。大気圏突入で失うなんて、遺憾ですからね。

-------

NO3- 10ppm
NO2- 0ppm
NH3 0ppm

窒素循環は安定。稚魚は順調、成長が待ちきれません。

close
　エントリー 5

またも軌道減衰に陥りました。原因は未だ不明。キャプテンに連絡する必要があります。アラバスターの航法システムに根本原因があるのなら、そうしないのは愚かでしょう。とはいえ、躊躇しています。私の疑念通り、もしこの件がジュニパーの過失だと明らかになれば、良からぬ事態が波及するかもしれません。一旦、軌道を再調整し、今のところ減衰は起きていません。

--------

「スマッシュ, マドル, スクラブ」は依然として謎です。

「消化不良」では、私は好成績を収め、勝利を確実にしました。今後はこれが私の優先希望だと主張するつもりです。

--------

キャプテンが航法問題の最新情報を持ち帰りました。このモデルの船に特有の先天的欠陥が原因とのことですが、なぜリコールへ至らなかったのか、全くもってわかりません。

私はどの時点でも、ジュニパーの怠慢だと非難したことはありませんが、この件は私たちの間に摩擦を作りました。昨日バーミリオンが間に入って話をしましたが、まだ緊張したままです。今回の新情報を踏まえ、私は彼女に歩み寄り、謝罪すべきと感じます。

--------

打ち上げまであと1週間強。ユニットの組み立ては完了。あとは5日間のストレス・テストと、手順確認のリハーサルを行う1日のみです。

まるで、私自身がストレス・テストを受けているような気分。

-------

NO2- 0ppm
NH3 0ppm

稚魚は引き続き順調。ベビー・ブラインシュリンプを夢中で食べています。孵化場の容量を増やす必要あり。

　エントリー 6

今日、ジュニパーが虫垂炎になりました。幸いなことに、彼女はバーミリオンの優秀な腕に委ねられています。もちろん、アラバスターの卓越した医療技術にも。たった一日か二日で回復し、活動できるようになるでしょう。
人がまだ地球に縛られていた、そう遠くない昔。虫垂炎は、何週間も寝たきりになる病でした。さらに遡れば、命を落とすこともあったなんて、想像を絶します。人類は本当に特別な存在です。この短い期間でどれだけの進歩を遂げたか考えると、殊更そう感じます。

この任務が、更なる躍進の一助となることを願っています。

-----

またゲームナイト、またナスの勝ち。「消化不良」の提案は無視されました。

-----

私が判断する限り、明日の打ち上げ準備は万端です。興奮していますが、緊張もしています。緊張はすべきではありません。そうなる理由もありません。
私はチームとしての自分たちに、全幅の信頼があります。打ち上げ手順の確認は励みになりました。あとは、早く寝る必要がありますね。

そう。それが私に必要なことです。ぐっすり眠ること。

-----

まあ、無理でしたね。5時間経ちましたが、まだ起きています。

-----

何が起きたのか、まだ理解できません。内部の何かを見落としたに違いありません。記録したデータはどれも、画面で見ていたものと相関しないようです。
テレメトリのデータを何度も確認しました。申告によると、探査機との接続を失ったのは、惑星表面より低い高度に達してからとのことでした。明らかなエラーですが、これらのシステムは打ち上げ前に繰り返しテストし、全く不備なく機能していました。

ジュニパーは自分を責めているようですが、彼女が原因ではありません。

振出しに戻る、ですか。

　エントリー 7

探査機の失敗について、キャプテンとTFCから厳しく追求されています。報告書を提出しましたが、彼らは独自の調査を進めているようです。しばらくは頭から追い出して、目の前の仕事に集中しようと思います。

明日から新しい探査機の開発に取りかかります。

------

今日はナス、ピスタチオと短時間会って、この放射線スパイクの潜在的な原因について話し合いました。2人とも専門ではありませんが、考えるべき仮説を提案してくれました。

------

今日、発見がありました。予想外のもの。アーノルド-475Mは、ホログラムです。あるいは、少なくともホログラム様の固有性をもっています。 ママレードは彼の言葉で「惑星の映像異常」を観測しました。以来、私もそれを目にしています。
探査機に何が起きたのか、それで説明がつきます。いえ、完全にではありませんが、なぜアーノルドの表面だったはずのところを通過していったのか、今では理解できます。その後、探査機はさらに下の構造物と衝突したに違いありません。

キャプテンに連絡しました。彼女は、この星系からの退去を求めるでしょう。場合によっては、ファーマメントに戻るようにと。私たちは優秀ですが、こういった状況に対処する資格を持つ者は誰もいません。

それに、この事実は任務を無意味なものにします。

　エントリー 8

惑星がホログラムである件について、スワンと話しました。彼女はファーマメントへの帰還を告げませんでした。星系からの撤退さえ。
それどころか、私たちにアーノルド-475Mの近傍に留まり、任務自体をその観測に変更するよう求めています。

驚いたと認めざるを得ません。彼女は私に対し、今までになく強硬でした。スワンとは認識が一致していると思っていましたが、今は疑問を抱いています。

何よりも船員の安全が第一です。彼女の命令は、これを脅かすものと感じます。

------

今日の打ち合わせは酷いものでした。船員は、このような未知の存在の隣人となることに不満を抱いています。とりわけジュニパーは率直に意見をぶつけました。

正直、彼女を責めることはできません。私たちは同意もしていないものの為に、命を危険に晒しているのです。二者択一のうちもう片方は、違法となる命令無視。

私は自分たちの意志よりも、仕事を重んじるの？ ことによると、命よりも？

-------

また、放射線スパイク。アーノルドについての情報を踏まえると、特に憂慮が必要です。これは私たちが対峙するものが、不活性な構造物ではないことを示唆します。

状況をスワンに伝え、船員の最善の利益のためアーノルドから距離を置くよう、再び暫定的に要請しました。

今日、彼女への最後の敬意を失いました。

-------

アーノルドの放射線出力が大きな懸念となっています。これで3度目の急激なスパイクです。深刻な健康リスクとなる域に急速に近づいています。でも、船員たちに伝えるのが怖いです。皆、とても神経質になっています。彼らを追い詰めてしまうのではないかと、私は恐れています。

そうですね、バーミリオンに相談を…

-------

ここを去るべきでした。これは、私の過ちです。
